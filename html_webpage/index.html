Semestrální projekt z předmětu B4M39GPU

Implementace algoritmu IDW a jeho vizualizace na GPU

Vladislav Trnka

Zadání

Idw je metoda interpolace pomocí zadných uzlových bodů a jejich známé funkční hodnoty. Tato práce se zabývá efektivní implementací dané metody na GPU a CPU. Finální výsledek je interaktivní vizualizace generovaného terénu, pomocí uživatelem definovaných uzlových bodů.

Ovládání

Po spuštění programu se do příkazové řádky vypíše veškeré ovládní.

Mouse functions
Left click : Add anchor point
Right click : Delete anchor point
Wheel up/down : Change value of added point

Keys functions
ESC : Close the program
f : Toggle fullscreen
h : Print this message
v : Toggle VSync - vertical synchronization - enables more fps than 60
g : Disable or enable CPU kernels / aka slow ones

ARROW UP : Increment p param
ARROW DOWN : Decrement p param
ARROW LEFT : Change palette to the right
ARROW RIGHT : Change palette to the left
TAB : Move to next IDW computation method

c : Clear all anchor points
r : Generate random anchor points

0 : Save current anchor points as user data
+ : Read user data

1 : Read data - 50 points / resolution 256x256
2 : Read data - 100 points / resolution 256x256
3 : Read data - 150 points / resolution 256x256
4 : Read data - 200 points / resolution 768x768
5 : Read data - 300 points / resolution 768x768
6 : Read data - 400 points / resolution 768x768
7 : Read data - 100 points / resolution 1920x1080
8 : Read data - 200 points / resolution 1920x1080
9 : Read data - 500 points / resolution 1920x1080
